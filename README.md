## Project Overview

**TaskMaster Pro** is a **single-page task management application** that emphasizes simplicity, usability, and productivity.

It includes:

- **Add, complete, and delete tasks** dynamically
- **Filter tasks** by All, Completed, or Pending
- Persistent storage using **localStorage**
- Responsive design for desktops and mobile devices
- Clean UI with **icons** and hover effects

---

## Features

- Fully responsive design
- Modern UI with **Font Awesome icons**
- Dynamic task management with **JavaScript**
- Persistent task storage in **localStorage**
- Filter tasks easily by status
- Smooth hover effects and completed task styling
